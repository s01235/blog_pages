虚拟向导（Virtual Guide）相关论文总结

Chenjian_Song

人机协作/相关论文

发布于：2021年10月29日

本文主为人机协作领域中关于虚拟向导的相关研究文章总结。并附有文章的引用链接及对应的PDF链接。

一、虚拟向导的设计

1. Robotica-Susana-2020

文章标题：Toward an intuitive and iterative 6D virtual guide programming framework for assisted human—robot comanipulation.

摘要：虚拟向导具有提高任务精度并避免环境等产生的干扰，但构建虚拟想到通常需要专家知识和对任务进行建模，从而限制了虚拟向导的应用。现有文献很少考虑到方向的约束，即便考虑了，也仅仅是对位移和方向进行单独考虑，并且两者的运动并没有同步起来。本文针对上述问题，提出了一种新的框架，并允许用户创建6维的虚拟向导，通过采用结合针对位移的$Akima\;splines$插值，以及针对方向四元数的球面三次插值（$spherical\;cubic\; interpolation\; of\; quaternions$）的方法。最终通过先示教位移轨迹，然后在位移路径的约束下示教方向轨迹。最终模拟一个工业场景并给出了一个直观的解决办法。

关键词：Virtual Guide

创新点：同时考虑位移空间和方向空间等6个自由度的约束。

二、虚拟向导的应用

1. RAL-Xingguang-Duan-2021

文章标题：Virtual-Fixture Based Drilling Control for Robot-Assisted Craniotomy: Learning From Demonstration

摘要：穿颅手术中最有保障的一种方法就是采用人类引导—机器人辅助结合手术，本文研究包括高精度协作钻孔中的对齐和钻孔任务，通过人类的交互力进行意图识别，进而有效的在对齐和钻孔进行切换。切换条件的训练依据高斯混合模型GMMs实现。意图的识别通过使用GMMS和人类实时的引导力的KL（ Kullback-Leibler）散度实现。未验证方法的正确性，做了两次实验，对应钻孔任务的各个步骤。

关键词：Virtual-Fixture, Learning From Demonstration;

创新点：1、提出了一种人机协作进行穿颅手术的解决方法。

2. Robotica-papageorgiou-2020

文章标题：Kinesthetic guidance utilizing dmp synchronization and assistive virtual fixtures for progressive automation

摘要：渐进自动化框架允许机器人在编程过程中通过演示离散运动任务，从动觉引导无缝过渡到自主操作模式。这是通过动态运动原语(dmp)、虚拟夹具和可变阻抗控制的协同作用实现的。提出的dmp编码演示轨迹，并与用户当前演示同步，使参考生成的运动跟随人的演示。提出的虚拟夹具帮助用户重复学习的运动学行为，但允许穿透，以便用户可以在需要时修改学习的轨迹。采用变刚度策略，利用跟踪误差与相互作用力和力矩相结合的方式来调整渐进自动化水平，实现人机主导作用的转移。利用一种能量罐方法来应用所设计的控制器，并证明了总体控制方法的无源性。对所提出的框架进行了实验评价，结果表明，该框架在少数实例中实现了向自主模式的过渡。

创新点：1、创新型的提出渐进自动化的概念；2、结合DMP编码演示轨迹与用户当前演示进行同步，虚拟夹具帮助用户重复学习的运动学行为。

3. TRO-Bischof-2018

文章标题：A path/surface following control approach to generate virtual fixtures

摘要：机器人的工作空间可以通过虚拟夹具进行限制，以辅助操作者进行物理的人机交互任务。本文将曲面跟随控制与柔度控制相结合，提出了一种系统生成虚拟夹具的路径/曲面跟随控制方法。这种方法允许实现许多类型的约束，如引导和禁止区域虚拟夹具，硬和软约束，以及静态和动态虚拟夹具及其组合。此外，给出了所提出的控制方案的闭环稳定性证明。

创新点：通过PFC/SFC的方法去产生虚拟向导。

4. RAL-Bazzi-2021

题目：A Unified Approach for Virtual Fixtures and Goal-Driven Variable Admittance Control in Manual Guidance Applications

摘要：在人机物理交互的应用中，不仅要保证末端执行器的位置精度，而且要保证末端执行器的姿态精度。平动的运动对人来说很容易理解，而旋转的运动则不能立即解释。这使得操作者在设定目标方位时无法获得足够的定位精度。通过制定旋转虚拟夹具，提出了一种适合于人完成旋转运动的策略。这是由于一个聪明的方向描述，允许绘制一个等价的公式化的平移虚拟夹具和他们的旋转对等物。为了进一步提高人们对旋转世界中最佳运动方向的理解，还设计了变导纳控制。最后，我们建议联合利用这些技术来实现最佳的定位精度。通过涉及多个对象和ABB IRB-140机器人的旋转点对点合作运动来评估性能。

创新点：通过制定面向任务位置的旋转的虚拟夹具，提高了人们对旋转世界中最佳运动方向的理解。

5. Pruks2018

题目：Preliminary study on real-time interactive virtual fixture generation method for shared teleoperation in unstructured environments.(非结构化环境中共享遥操作实时交互式虚拟夹具生成方法初步研究)

摘要：提出了一种非结构化远程环境中交互式生成共享远程操作虚拟夹具的方法。该方法允许操作者在运动中直观地分配各种类型的虚拟夹具，以提供虚拟引导力，帮助操作者完成给定任务，同时最小化认知工作量。该方法利用从机器人底座旁边的深度和颜色视频传感器自动提取的几何特征（形状、表面等），增强从机器人摄像机的视觉反馈图像。人类操作员可以在计算机屏幕上选择一个特征，然后自动将其与虚拟触觉夹具关联。通过一个钉孔任务对该方法的性能进行了评估，实验表明该方法在遥操作性能上有了改进。
